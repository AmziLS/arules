Getting Started

Installing the ARulesXL Add-in

After installation start Excel. It should startup normally with the addition of an 'ARulesXL' menu.

If you do not see the menu item, select Tools | Add-Ins from the Excel menu and make sure the 'ArulesXL' box appears on the list of add-ins and is checked.

If ARulesXL does not appear on the list of add-ins, perhaps it was installed into another version of Excel on your PC. If that is not the case, contact us for assistance.

Using the Runtime

You can use the runtime whether or not you have a license.

Start Excel and open the sample Loan Approval.xls. It's in the ARulesXL install directory which defaults to: Program Files/amzi/ARulesXL/Samples.

You can type inputs into the green cells. Change the credit rating from 700 to 600. Notice that the status in the blue output cells below now says the credit score is too low and the loan is denied. Change the credit rating to a number greater than 700 and it's approved.

You can experiment with the other input fields as well, and read the rules on the right. But notice that if you try to edit the rules, you are prevented. That's because you can't change the rules in runtime only mode.

Activating your ARulesXL License

To activate the software, open the Start menu, select ARulesXL and run 'activate'. It will ask for your name for an individual license, or your organization's name for an enterprise license, and the reference number of the sale. You will most likely see that you have activated the software succesfully, but you might also find that a key is to be e-mailed to you instead. Once you receive it, you can run 'activate' again and enter the key directly in the dialog box.

Getting Started Step-by-Step

  1. Start Excel
  2. Rules must be part of a rule set. To create a rule set, select a range of cells, A1:A4, then select ARulesXL | New... enter PriceRules and click OK.
  3. A1 will have the name PriceRules. Type these rules into the cells A2 and A3:
    UnitPrice = 5 WHEN Quantity < 10
    UnitPrice = 3 WHEN Quantity >= 10
  4. Now you have a simple rule set that is ready to be queried. Select cell A6, outside of the rule set, and enter:
    5. = RQuery(PriceRules, "FIND UnitPrice when Quantity = 12" )
    The value of the cell should be 3
  5. What if you want to use a value from another cell in your query? To do this, in cell A5 enter the value 5.
  6. Change A6 to: = RQuery(PriceRules, "FIND UnitPrice whenQuantity = _1 ", A5)
    The value of the cell should be 5

Using ARulesXL

After installation you should have a new 'ARulesXL' menu under your 'start' programs. This menu has entries for the tutorial spreadsheets, documentation and sample spreadsheets. We recommend you start by:

  • Reading the Introduction to Rule Technology
  • Following the ARulesXL Tutorial
  • Look at the sample spreadsheets 'Techniques Basic.xls' and 'Techniques Advanced.xls' for insights into writing rules. To open these samples select 'Samples' from the ARulesXL menu in Excel or on the Windows Start Menu.

If you have questions or problems please visit contact us or visit our user forum.

Uninstalling ARulesXL

To remove ARulesXL, first close Excel.

Then, on your Windows Start Menu open the 'Control Panel' and select 'Add/Remove Programs'. Highlight ARulesXL and click 'Remove'.

You should uninstall the old version of ARulesXL before installing a new one.

If you are getting an error message about not finding ARulesXL.xla or arules.dll, in Excel, select Tools | Add-Ins. Then uncheck the box next to 'ARulesXL' and click OK.
