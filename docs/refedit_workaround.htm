If you get this error trying to install ARulesXL, you must:

  • Open Developer Mode (Alt-F11 if no menu item)
  • Open Tools / References
  • You'll see a missing reference for RefEdit, uncheck it
  • Close the references dialog
  • Open it again, and select 'browse'
  • Find refedit.dll, it will be in the Office11 or Office12 directory under Program Files \ Microsoft Office
  • Check it has a reference
  • Recompile ARulesXL.xla using Debug / Compile ARulesXL.xla

Our apologies for the inconvenience.
