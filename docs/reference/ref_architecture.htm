Product Architecture

ARulesXL is a full business rule engine integrated with Excel's spreadsheet engine. The integration is implemented as an Excel Add-In which allows the rule engine to share Excel's spreadsheet user interface.

The files that comprise ARulesXL are:

  • arules.* - The ARules rule engine.
  • arulesxl.xla - The Excel Add-In integrating the ARules engine with Excel.
