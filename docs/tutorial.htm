There are three tutorials for ARulesXL. One introduces the features of ARulesXL, another introduces rule design, and the third is multi-sheet workbooks with graduated examples on each worksheet.

A Quick Tour of the ARulesXL Excel Environment

See how inputs, outputs and rules are linked together in this brief overview of ARulesXL rule development.

Learning ARulesXL

This tutorial walks you through the basic ARulesXL concepts explaining how to create and use rules and rule sets.

Designing Rules and Rule Sets

The Designing Rules Tutorial focuses on knowledge engineering, the process of analyzing knowledge and converting it into rules that can be used on a spreadsheet. It uses a simple product advisor as its implementation example.

Tables, Collections, Variables and Loops

The Tables, Collections Variables and Loops demonstrates a variety of useful techniques for working with tables and collections of data. Working samples for each technique are provided in the 'collections' folder in the tutorials.

Techniques Spreadsheets

The two sample spreadsheets Techniques Basic.xls and Techniques Advanced.xls have multiple worksheets that introduce ARulesXL concepts one sheet at a time. The first sheets have very basic rule sets and they build to more complex examples, introducing new concepts on each successive sheet.
