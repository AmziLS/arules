Given a table of input data, and a required output for each row, you can add a column to the table that has an RQuery() which takes it's inputs from the table values in that row as described in the background section.

Simply copying that formula down the rest of the column (Excel will update the cell references for each row) provides a report with each loan's resaleability given in the last column.
